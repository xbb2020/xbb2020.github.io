# 索引

假设字符串长度为 N
则下标 0 到 N-1 为正向索引
- 1 到 - N 为反向索引
s1 [N-1] 和 s1 [-1] 都是字符串的最后一个字符

# 切片

s1 [2:5]：下标 2 到 5 的正向切片
s1 [-7:-4]：下标 - 7 到 - 4 的正向切片
s1 [2:]：下标 2 到最后的正向切片
s1 [-7:]：下标 - 7 到最后的正向切片
s1 [2::2]：下标 2 到最后，间隔一个字符取一次的正向切片
s1 [7:1:-1]：下标 7 到 1 的负向切片
