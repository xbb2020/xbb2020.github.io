头指针的作用：使空表和非空表的操作统一

  • 头插法：直接在头指针后插入新元素
    1. 建立要插入的元素节点
    2. 新节点 next 指向头指针的 next
    3. 头指针 next 指向新节点
  • 尾插法
    1. 建立一个指针指向链表最后一个指针
    2. 新建元素节点
    3. 尾指针 next 指向新节点
    4. 尾指针指向新节点

一直 next，直到符合条件
注意判断空指针

  • 前插：
    1. 新建元素节点
    2. 找到插入位置节点的前驱节点
    3. 新节点 next 指向前驱的 next
    4. 前驱 next 指向新节点
  • 后插：
    1. 新建节点
    2. 找到插入位置节点
    3. 新节点 next 指向插入节点 next
    4. 插入节点 next 指向新节点
  1. 找到要删除的节点的前驱
  2. 前驱节点的 next 指向被删除节点的 next
  3. 如果题目要求释放被删除节点，则需要用变量保存被删除节点，再用 free () 来释放内存

int binarySearch(ElemType a[], KeyType k, int low, int high){
    if(low <= high){
        int mid = (low + high) / 2;
        if(a[mid].key == k)return mid;
        else if(k < a[mid].key) return binarySearch(a, k, low, mid-1);

# 排序

  • 直接插入

    解释：固定第一个元素，从第二个元素开始，逐渐向前搜索，直到找到它的位置时把他插入，其他元素在向前搜索的过程中已经后移
    时间复杂度：O (n2n^2n2)
    稳定的排序方法

  • 冒泡

    解释：从最后一个元素开始，向前逐渐比较相邻元素，逐渐使最值移动到最前面，再比较时前 i 项已经有序且为最值，因此不需要比较。若 swap 变量为 false，则表示没有发生交换，此时序列已经有序，排序结束
    时间复杂度：O (n2n^2n2)
    稳定

  • 选择

    解释：遍历找到最值，放到前面，再在剩余值里找最值，循环 n-1 次，最后一个值就不用看了
    时间复杂度：O (n2n^2n2)
    不稳定

  • shell


    解释：将待排序序列分割成若干个子序列（每个序列由相隔增量的元素组成），子序列分别进行直接插入排序，逐渐缩小增量到 1
    时间复杂度：O (n1.25n^1.25n1.25)
    不稳定

  • 快排


    解释：选取一个值作为基准，从两侧开始，比它小的数放到左侧，比它大的数放到右侧，再把它放到最后的空位上，此时基准值位置固定，两侧分成两个子序列，继续进行这个过程
    时间复杂度：O (nlog2nnlog_2nnlog2​n)
    不稳定
